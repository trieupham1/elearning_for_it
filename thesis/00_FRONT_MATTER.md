## TÓM TẮT (ABSTRACT - VIETNAMESE)

Với sự phát triển mạnh mẽ của công nghệ thông tin và nhu cầu chuyển đổi số trong giáo dục, việc xây dựng một hệ thống quản lý học tập trực tuyến hiện đại là vô cùng cần thiết. Đồ án này trình bày việc thiết kế và xây dựng một nền tảng E-Learning hoàn chỉnh dành riêng cho ngành Công nghệ Thông tin.

Hệ thống được phát triển với kiến trúc Client-Server, sử dụng Flutter Framework cho ứng dụng di động đa nền tảng (Android, iOS, Web), kết hợp với backend Node.js/Express và cơ sở dữ liệu MongoDB. Hệ thống hỗ trợ ba vai trò người dùng chính: Quản trị viên (Admin), Giảng viên (Instructor) và Sinh viên (Student).

**Các tính năng chính bao gồm:**
- Quản lý khóa học, học kỳ và người dùng
- Hệ thống bài tập với tự động chấm code (Python, Java, C++, JavaScript, C)
- Quiz trắc nghiệm tự động với ngân hàng câu hỏi
- Điểm danh QR Code với xác thực GPS
- Video bài giảng với theo dõi tiến độ học
- Forum thảo luận và chat realtime
- Thông báo đa kênh (In-app + Email)
- Dashboard thống kê và báo cáo
- Video call trực tiếp trong khóa học

Hệ thống đã được triển khai thành công trên nền tảng cloud (Render.com) và đạt được các mục tiêu đề ra về tính năng, hiệu suất và trải nghiệm người dùng.

**Từ khóa:** E-Learning, Flutter, Node.js, MongoDB, Auto-grading, Video Streaming, Real-time Communication

## ABSTRACT (ENGLISH)

With the rapid development of information technology and the need for digital transformation in education, building a modern online learning management system is essential. This thesis presents the design and development of a complete E-Learning platform specifically for Information Technology education.

The system is developed with a Client-Server architecture, using Flutter Framework for cross-platform mobile applications (Android, iOS, Web), combined with Node.js/Express backend and MongoDB database. The system supports three main user roles: Administrator (Admin), Instructor, and Student.

**Key features include:**
- Course, semester, and user management
- Assignment system with automatic code grading (Python, Java, C++, JavaScript, C)
- Automated multiple-choice quizzes with question bank
- QR Code attendance with GPS verification
- Lecture videos with progress tracking
- Discussion forum and real-time chat
- Multi-channel notifications (In-app + Email)
- Statistical dashboard and reporting
- Direct video calls within courses

The system has been successfully deployed on cloud platform (Render.com) and achieved the set objectives for features, performance, and user experience.

**Keywords:** E-Learning, Flutter, Node.js, MongoDB, Auto-grading, Video Streaming, Real-time Communication
